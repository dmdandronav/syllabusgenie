# Unit 3 Notes — Thermodynamics

## Hess's Law
You can add chemical equations to get an overall equation. The ΔH values add the same way.
Example: If A→B has ΔH₁ and B→C has ΔH₂, then A→C has ΔH = ΔH₁ + ΔH₂

## Entropy Rules of Thumb
- More moles of gas = higher entropy
- Dissolution usually increases entropy
- Temperature increase = entropy increase

## Common Mistake
Students confuse exothermic (ΔH < 0, releases heat) with spontaneous. A reaction can be
exothermic AND non-spontaneous if entropy decreases enough.
