# SyllabusGenie
Drop in your class syllabus, notes, and past worksheets — get a study coach that only knows your actual material, not generic answers.

## Stack
- Frontend: React + Vite + Tailwind v4
- Backend: Flask + OpenAI-compatible API + RAG

## Quick start
1. Add your class materials as .txt or .md files to backend/data/
2. cd backend && pip install -r requirements.txt && python app.py
3. cd frontend && npm install && npm run dev
